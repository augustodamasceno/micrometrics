# micrometrics
Precision profiling and behavioral analysis suite for fundamental language constructs and data structure scenarios.

# Directory Structure

The format is: `language/number-name/`  

* **`language`**: The programming language used (e.g., `Cpp`, `C`, `Python`).  
* **`number`**: A sequential identifier for indexing.  
* **`name`**: A concise name for what is being tested.  

With

* `language/number-name/results/number-name.md`: for results outputs and visualizations.  
* `language/README.md`: with build instructions.

# Online Compilers & Editors

Websites to compile and run code directly from the browser, organized by language.

## C++
- [Compiler Explorer (godbolt.org)](https://godbolt.org) — inspect assembly output across many compilers and versions
- [Wandbox](https://wandbox.org) — multiple C++ standards and compilers (GCC, Clang)
- [Coliru](https://coliru.stacked-crooked.com) — quick snippets with GCC
- [cpp.sh](https://cpp.sh) — simple online C++ shell

## C
- [Compiler Explorer (godbolt.org)](https://godbolt.org) — supports C alongside C++
- [OnlineGDB](https://www.onlinegdb.com/online_c_compiler) — C compiler with debugger
- [Programiz C Compiler](https://www.programiz.com/c-programming/online-compiler/) — beginner-friendly

## Python
- [Python.org Online Shell](https://www.python.org/shell/) — official CPython REPL
- [Replit](https://replit.com) — full Python environment with package support
- [PythonTutor](https://pythontutor.com) — visualizes execution step by step
- [Google Colab](https://colab.research.google.com) — notebooks with GPU/TPU support

## Java
- [Compiler Explorer (godbolt.org)](https://godbolt.org) — Java bytecode and JIT output
- [OnlineGDB](https://www.onlinegdb.com/online_java_compiler) — Java compiler with debugger
- [Replit](https://replit.com) — full Java environment
- [JDoodle](https://www.jdoodle.com/online-java-compiler/) — multiple JDK versions

## JavaScript / TypeScript
- [JSFiddle](https://jsfiddle.net) — HTML/CSS/JS playground
- [CodePen](https://codepen.io) — front-end focused playground
- [TypeScript Playground](https://www.typescriptlang.org/play) — official TS compiler with type checking

## Rust
- [Rust Playground](https://play.rust-lang.org) — official Rust playground
- [Compiler Explorer (godbolt.org)](https://godbolt.org) — assembly output for Rust

## Go
- [Go Playground](https://go.dev/play/) — official Go playground
- [Compiler Explorer (godbolt.org)](https://godbolt.org) — Go assembly output

## Multi-language
- [Compiler Explorer (godbolt.org)](https://godbolt.org) — C, C++, Rust, Go, Java, Haskell, and many more; ideal for assembly-level analysis
- [Replit](https://replit.com) — supports 50+ languages with a full IDE experience
- [OnlineGDB](https://www.onlinegdb.com) — C, C++, Java, Python, Go, and others with an integrated debugger
- [JDoodle](https://www.jdoodle.com) — 70+ languages
- [Ideone](https://ideone.com) — 60+ languages, useful for quick pastes
